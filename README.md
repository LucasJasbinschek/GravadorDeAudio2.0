# Audio Recorder Extension

Extensão criada para gravação de áudio e geração de link compartilhável utilizando a API do tmpfiles.org.

## 🚀 Funcionalidades

- Gravação de áudio diretamente pelo navegador
- Geração automática de link do áudio gravado
- Upload utilizando API do tmpfiles.org
- Interface simples e rápida de utilizar

## 🛠 Tecnologias Utilizadas

- HTML
- JavaScript

## 📦 Instalação

1. Baixe o arquivo `.zip` do projeto
2. Descompacte os arquivos em uma pasta
3. Abra o navegador e acesse a área de extensões
4. Habilite o **Modo Desenvolvedor**
5. Clique em **Carregar sem compactação**
6. Selecione a pasta do projeto descompactado
7. A extensão ficará disponível na parte superior do navegador

## ▶️ Como Utilizar

1. Clique no ícone da extensão
2. Inicie a gravação do áudio
3. Finalize a gravação quando desejar
4. Clique em gerar link
5. Aguarde a geração do link
6. Copie e compartilhe o link gerado
